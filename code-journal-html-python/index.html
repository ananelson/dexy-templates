This is a template for a simple python code journal, written in HTML.

The 'journal.html' file contains your journal content. The _header.html file takes care of doctypes and CSS, and the _footer.html file closes the HTML tags that _header.html opens, so you can just start writing your actual content in journal.html. If you get bored writing paragraph tags, you can also write your journal in markdown.

This template uses the atatonic web framework to give nice typography and a simple, clean look.
